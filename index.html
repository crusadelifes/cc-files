Setup Guide

// First-time configuration walkthrough

1

Install Dependencies

Run this command in your terminal or hosting panel's console before starting the bot.

pip install discord.py aiosqlite aiohttp python-dotenv
2

Get Discord Bot Token

Go to discord.com/developers/applications → New Application → Bot → Reset Token. Enable ALL 3 Privileged Gateway Intents (Presence, Server Members, Message Content).

TOKEN = "your_token_here" # Already set in bot.py
3

Get TMDB API Key (Free)

Sign up at themoviedb.org → Settings → API → Create → Developer. Paste your key using the command below.

-settmdb your_tmdb_api_key_here
4

Invite Bot to Server

In Developer Portal: OAuth2 → URL Generator → Check "bot" and "applications.commands" → Bot Permissions: Administrator → Copy URL and open in browser.

5

First Commands (Run as Server Owner)

Run these in order after the bot comes online. This unlocks all admin features for your team.

-setbotmanager @YourAdminRole -setlogchannel #logs -setmodlogchannel #mod-logs -setwelcome #welcome Welcome {user} to {server}! -setautorole @Member -setcurrency 🎬 Credits -settmdb your_tmdb_key
6

Start Using the Bot

The bot is now ready! All commands use the - prefix. Type -help in Discord anytime for a quick reference.

-help # Quick command overview -level # Check your level -balance # Check your balance -daily # Claim daily reward
